# Live Demo
Try it! [Imposible Hangman ](https://imposiblehangman.netlify.app/)

# Getting Start
## Setup
* git clone https://github.com/sevillecarlos/imposible-hang-man.git or Download the zip.
### In the terminal enter 
* cd /imposible-hang-man
* npm install
* npm run start
